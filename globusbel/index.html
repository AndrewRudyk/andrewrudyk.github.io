Globe Of Belarus (Глобус Беларуси)

Aug 10, 2017
Attractions of Belarus with photos and descriptions (website source: globus.tut.by). Navigation is by map or search tabs

App was in AppStore with august 2017 to august 2018

Main features and frameworks:

- Swift
- text and XML parsing uses JSONSerialization
- Google Maps API for showing and navigating maps
- CoreData is used
other frameworks: CoreLocation, Alamofire and etc
